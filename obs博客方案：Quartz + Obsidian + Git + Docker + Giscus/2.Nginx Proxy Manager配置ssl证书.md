到了这一步，你只需要**在 Nginx Proxy Manager 的网页管理后台里，通过点选和填写表单来完成配置**，不需要写任何代码或操作命令行。整个流程主要分为登录后台、添加代理规则、申请SSL证书三步。

### 🛠️ 第一步：登录 Nginx Proxy Manager 管理后台

1. 打开你的浏览器，访问 `http://<你的服务器IP>:81`。
2. 使用默认的登录凭证登录 [\-1](https://cloud.baidu.com/article/4069390)[\-2](https://itrustssl.cn/column/wiki/212.html)[\-3](https://developer.aliyun.com/article/1639807)：
	- **邮箱**：`admin@example.com`
	- **密码**：`changeme`
3. 首次登录后，系统会**强制要求你修改默认的邮箱和密码**，请务必牢记你设置的新密码 [\-3](https://developer.aliyun.com/article/1639807)[\-8](https://cloud.tencent.cn/developer/article/2440711?from=15425)。

### ➕ 第二步：添加代理主机 (Proxy Host)

这一步的目的是告诉 NPM，当用户访问你的博客域名时，应该把请求转发到你服务器上运行的 Quartz 容器。

1. 登录成功后，点击顶部导航栏的 **`Hosts`**，然后选择 **`Proxy Hosts`** [\-3](https://developer.aliyun.com/article/1639807)[\-8](https://cloud.tencent.cn/developer/article/2440711?from=15425)。
2. 点击右上角的 **`Add Proxy Host`** 按钮。
3. 在弹出的表单中，填写以下关键信息 [\-3](https://developer.aliyun.com/article/1639807)[\-8](https://cloud.tencent.cn/developer/article/2440711?from=15425)：
	- **Domain Names**：填写你为博客准备的**域名**（例如 `blog.yourdomain.com`）。**重要：** 在这之前，你需要先去你的域名服务商（如腾讯云DNSPod）后台，将你的域名通过 **A 记录** 解析到你服务器的公网 IP 地址上 [\-3](https://developer.aliyun.com/article/1639807)。
	- **Scheme**：保持默认的 `http` 即可。
	- **Forward Hostname / IP**：填写你服务器的内网 IP 地址。由于 NPM 和你的 Quartz 博客都运行在同一台服务器的 Docker 中，通常可以填写 `172.17.0.1`（这是 Docker 默认的网关地址）或你的服务器内网 IP（如 `10.0.0.4`）[\-3](https://developer.aliyun.com/article/1639807)[\-4](https://cloud.tencent.com.cn/developer/article/2527035?policyId=1003)。
	- **Forward Port**：填写 `8080`（这是你在 `docker-compose.yml` 中为 Quartz 容器映射的端口）。
	- 其他选项如 `Cache Assets`、`Block Common Exploits` 可以根据你的需求勾选，不勾选也不影响基本功能。
4. 点击 `Save` 保存。此时，你应该可以通过 `http://你的域名` 访问你的博客了。下一步是配置 HTTPS。

### 🔒 第三步：申请 SSL 证书（Let's Encrypt）

这是最关键的一步，NPM 会自动帮你申请证书，并每三个月自动续期 [\-1](https://cloud.baidu.com/article/4069390)[\-8](https://cloud.tencent.cn/developer/article/2440711?from=15425)。

1. 在刚才保存的代理主机列表中，点击你刚创建的那条记录的 **`Edit`**（通常是三个点的菜单）[\-3](https://developer.aliyun.com/article/1639807)。
2. 切换到 **`SSL` 标签页**。
3. 在 `SSL Certificate` 下拉菜单中，选择 **`Request a new SSL Certificate`** [\-3](https://developer.aliyun.com/article/1639807)。
4. 勾选 **`Force SSL`**，这样所有访问你网站的 HTTP 请求都会被自动重定向到 HTTPS [\-3](https://developer.aliyun.com/article/1639807)[\-8](https://cloud.tencent.cn/developer/article/2440711?from=15425)。
5. 填写你的 **邮箱地址**（用于接收证书过期等通知），并勾选同意 **Let's Encrypt 的服务条款** [\-1](https://cloud.baidu.com/article/4069390)[\-3](https://developer.aliyun.com/article/1639807)。
6. 点击 `Save` 保存。

等待几秒钟，页面刷新后，你的 SSL 证书就申请并配置好了。现在，你可以通过 `https://你的域名` 安全地访问你的 Quartz 博客了，地址栏旁会出现代表安全的“小锁”图标。

如果在填写`Forward Hostname/IP`时不确定具体值，或者在开启`Force SSL`后遇到访问问题，随时告诉我，我帮你看看。
